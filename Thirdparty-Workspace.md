## Download Source Code Workspace
Create the following directory and move into it.  

    mkdir -p ~/val_indigo/src
    cd ~/val_indigo/src
    catkin_init_workspace

Pull down any third party code here.  

Source your completed environment.  

    source ~/.bashrc
